# Lyrics Cloud (Offline)

私有离线中文歌词检索与可视化应用，支持 JSON/TXT/DOCX 离线导入、IndexedDB 持久化、搜索与沉浸式词云可视化。

## 功能概览

- 离线导入：支持 JSON/TXT/DOCX，多文件同时导入
- 本地持久化：IndexedDB 保存，刷新后仍可检索
- 搜索与筛选：标题/歌词/专辑/词作者检索，专辑/年份/词作者筛选
- 词云探索：高频词可视化，点击词语聚焦并显示相关歌词
- 词频面板：Top 词频柱状图展示
- 单字开关：可切换是否显示单字词
- 去重统计：重复歌词只计一次（词频）
- 中文优先分词：过滤英文/虚词噪声
- 侧边抽屉：导入/列表在右侧抽屉中展开，移动端自适应底部抽屉
- 主题切换：暖 / 墨绿 / 银青 / 骨蓝

## 启动

```bash
npm install
npm run dev
```

打开 `http://localhost:3000`。

## 导入歌词

点击右侧「导入」面板按钮，选择文件导入。支持多文件同时导入，数据会写入 IndexedDB，刷新仍可使用。

支持格式：
- JSON（结构见下）
- TXT/DOCX（自定义歌词文档格式）

TXT/DOCX 格式规则（按行解析）：
- 专辑行：`《专辑》2015 歌手`（年份/歌手可省略）
- 标题行：`<歌名>`（支持中文括号样式）
- 元数据：`作词：...`（可同一行带 `作曲：...`，作曲暂不入库）
- 歌词正文：从元数据结束后到下一首标题/专辑行之前，保留换行

### JSON 示例

点击页面上的“导入歌词 JSON”，选择本地文件。数据会被写入 IndexedDB，刷新页面仍可读取。

示例结构：

```json
{
  "schema_version": 1,
  "artist": "演示艺人",
  "tracks": [
    {
      "title": "雾灯之外",
      "album": "回声档案",
      "year": 2024,
      "lyricists": ["示例词作者"],
      "lyrics": "这里是占位示例歌词文本..."
    }
  ]
}
```

> 注意：请勿导入真实版权歌词内容。
